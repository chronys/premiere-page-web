les displays

Display: block

Cette div est un element en Block. Il occupe toute la largeur de l'écran

Exemple de balises de type 'block' :

  • div
  • p
  • main
  • h1,h2,h3...,h6
  • li
  • ul

Display : Inline

Les elements en Inline ne forcent pas le retour à la ligne, il prend simplement la place dont il a besoin

attention !! les éléments de type inline ne sont pas comme les type block, leur hauteur et largeur ne sont délimités que par leur contenu

exemple de balises de type Inline

  • balise a (lien)
  • img
  • input
  • video
  • label
  • span

Display : Inline-block

Inline-block: C'est un type inline qui conserve les propriétés du type block, il peut donc être modifié en hauteur et largeur comme n'importe quel type block

Display: Flex

Flex : les éléments dans le conteneur s'alignent en ligne et se repartissent l'espace disponible

Display : grid

Grid : Avec grid, on peut definir une grille avec des lignes et des colonnes

Display: none

None: Le display none cache tout simplement l'élément.
